Lab 1: HTML and CSS

Part 1: HTML

  1. Make a copy of this html file and name it lab1.html. You will be modifying your copy of this file. While you are working on it, you will want to keep open this file, because it has links to some useful web pages.
  2. While you are reading through the source of this file, note the use of semantic elements such as section, article, and nav. It is best practices to use semantic elements where ever possible rather than divs. One important reason to do so is that it gives accessibility tools such as screen readers more information to make reading your page a better experience.
  3. Add a navigation menu (using the nav element) to lab1.htmlthat includes three items: anchor link to "Part 1: HTML", a fragment link to "Part 2: CSS", and a real link with the label "HTML Reference" and the link "https://developer.mozilla.org/en/docs/Web/HTML/"
  4. Add a 2-by-4 HTML table after the end of this list, where the first column is the name of each of the four pseudo-classes on the anchor element, and the second column is the two or three word description of the pseudo-class (or state).

Part 2: CSS

Add some style! You can choose colours, fonts and other things to style.

  1. Add all CSS in a file called styles.css and link it in your lab1.html document.
  2. Experiment with different styles for the table.
  3. Add styles for the anchor tags for each of the four pseudo-classes.
  4. Make the navigation menu appear on the left side of the content. You may pick one of several approaches to do this:
    • Basic using floating list items
    • If you have more CSS knowledge, make the menu responsive using @media queries and no Javascript.
  5. If you have prior CSS experience, try learning something new such as flexbox, css animations, or something else you thought you might want to try.
